### 11 种视觉风格

| 风格 | 描述 |
|------|------|
| cute (默认) | 甜美、可爱、少女风 - 经典小红书美学 |
| fresh | 清新、自然、干净 |
| warm | 温暖、友好、亲切 |

| minimal | 极简、精致 |

| chalkboard | 黑板+彩色粉笔、教育风 |
| study-notes | 真实手写笔记照片风格（蓝笔+红批注+黄荧光笔） |
| screen-print | 海报艺术风、半调纹理、有限配色、象征叙事 |

### 8 种布局

| 布局 | 描述 |
|------|------|
| sparse (默认) | 信息最少、冲击力最大（1-2个要点） |
| balanced | 标准内容布局（3-4个要点） |
| dense | 高信息密度、知识卡片风（5-8个要点） |
| list | 列表/排行榜格式（4-7条） |
| comparison | 左右对比布局 |
| flow | 流程/时间线布局（3-6步） |
| mindmap | 中心放射式思维导图（4-8分支） |
| quadrant | 四象限/圆形分区布局 |

- 可生成 2-10 张系列图片，自动保持视觉一致性
