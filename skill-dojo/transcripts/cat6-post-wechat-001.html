### baoyu-post-to-wechat 技能支持的操作

该技能用于发布内容到 微信公众号，支持两种主要发布类型：

---

### 📝 1. 文章发布（Article Posting）

支持三种输入类型：
- HTML 文件
- Markdown 文件
- 纯文本（自动转为 Markdown）

两种发布方式：

| 方式 | 特点 | 要求 |
|-----|------|-----|
| API (推荐) | 速度快 | 需要 API 凭证 (AppID/AppSecret) |
| Browser | 速度慢 | 需要 Chrome + 登录会话 |

核心功能：
- 4 种主题样式 (default, grace, simple, modern)
- 13 种颜色预设 + 自定义 hex 色值
- 自动生成标题/摘要
- 外部链接自动转底部引用（微信友好）
- 评论控制（开启/关闭、仅粉丝）
- 多账号支持

---

### 🖼️ 2. 图文发布（Image-Text Posting）

短内容 + 多图片（最多 9 张）：

# 通过 Markdown + 图片目录
--markdown article.md --images ./images/

# 直接指定标题/内容/图片
--title "标题" --content "内容" --image img.png --submit
