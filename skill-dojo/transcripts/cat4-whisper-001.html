### Whisper 语音转文字技能使用说明

openai-whisper 技能存在于系统中，用于本地语音转文字（无需 API Key）。

#### 📋 基本信息
| 项目 | 内容 |
|------|------|
| 技能名称 | openai-whisper |
| 功能 | 本地语音转文字 (Speech-to-Text) |
| 依赖 | whisper CLI 命令行工具 |
| 安装方式 | brew install openai-whisper |

#### 🚀 使用方式

1. 音频转文字（基本用法）
whisper /path/audio.mp3 --model medium --output_format txt --output_dir .


2. 音频翻译（自动翻译为英文）
whisper /path/audio.m4a --task translate --output_format srt


#### ⚙️ 关键参数
| 参数 | 说明 |
|------|------|
| --model | 模型大小（tiny/base/small/medium/large/turbo），默认 turbo |
| --output_format | 输出格式（txt/srt/vtt/json/all） |
| --output_dir | 输出目录 |
| --task | 任务类型：transcribe（转录）或 translate（翻译成英文） |

#### 📝 注意事项
- 首次运行时模型会自动下载到 ~/.cache/whisper
- 小模型速度快，大模型精度高
- 支持的音频格式：mp3、m4a、wav 等常见格式
