## 天问监控查询技能使用方式

### 1. 触发词
'天问'、'监控曲线'、'监控数据'、'Tianwen'、'查监控'、'监控大盘'

### 2. 适用场景
- 用户提供天问监控大盘链接，需要查询指标数据
- 需要查看某个监控指标的当前值、同比变化（vs昨日/vs7日前）

### 3. 核心能力
- URL解析：自动从天问URL中提取 dashboardId 和 panelId
- 数据获取：通过 Playwright 打开页面，拦截 graph/query 接口获取完整时序数据
- 登录支持：支持短信验证码、KIM Code扫码、邮箱密码三种登录方式

### 4. 输入要求
| 输入 | 必需 | 说明 |
|------|------|------|
| 天问监控URL | ✅ | 包含 dashboardId 和 panelId 的完整链接 |
| 手机号 | 推荐 | 11位手机号，用于短信验证码登录 |

### 5. 输出内容
{
  "指标名称": "xxx",
  "panelId": 123456,
  "统计": [
    {
      "曲线名称": "xxx",
      "当前均值": 152.05,
      "当前值": 179.8,
      "vs昨日均值%": -15.9,
      "vs昨日当前%": -17.7,
      "vs7日均值%": 6.4,
      "vs7日当前%": -0.5
    }
  ]
}


### 6. 典型交互流程
1. 用户提供天问URL → 技能解析 panelId
2. 检查登录状态（已有session则免登录）
3. 无session时收集手机号 → 发送短信验证码
4. 用户提供验证码 → 完成登录
5. 获取数据并展示汇总统计表
