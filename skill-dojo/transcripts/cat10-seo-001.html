## SEO审查核心检查项

### 一、技术SEO（Technical SEO）

#### 1. 可爬取性（Crawlability）
- Robots.txt：检查是否意外阻止了重要页面
- XML Sitemap：是否存在、可访问、格式正确、已提交到 Search Console
- 网站架构：重要页面是否在3次点击内可达、无孤立页面
- 爬取预算：参数化URL、分面导航、无限滚动是否处理得当

#### 2. 索引状态（Indexation）
- 使用 site:domain.com 检查索引量
- 检查是否有 noindex 标签误用、canonical 指向错误、重定向链/循环、软404

#### 3. 网站速度与核心Web指标
| 指标 | 标准 |
|------|------|
| LCP (最大内容渲染) | < 2.5秒 |
| INP (交互响应延迟) | < 200ms |
| CLS (累计布局偏移) | < 0.1 |

#### 4. 移动友好性
- 响应式设计、触摸目标大小、视口配置、无水平滚动

#### 5. 安全性与HTTPS
- 全站HTTPS、有效SSL证书、无混合内容

---

### 二、页面SEO（On-Page SEO）

| 检查项 | 要点 |
|--------|------|
| Title标签 | 唯一、主关键词靠前、50-60字符、吸引点击 |
| Meta描述 | 唯一、150-160字符、含关键词、有行动号召 |
| 标题结构 | 每页一个H1、层级逻辑清晰（H1→H2→H3） |
| 图片优化 | 描述性文件名、Alt文本、压缩体积、WebP格式 |
| 内部链接 | 重要页面链接充足、锚文本描述性、无断链 |

---

### 三、内容质量（Content Quality）

#### E-E-A-T 信号
- Experience（经验）：展示第一手经验、原创数据
- Expertise（专业性）：作者资质可见、信息准确详细
- Authoritativeness（权威性）：行业认可、被他人引用
- Trustworthiness（可信度）：信息准确、联系方式可见、隐私政策完整

---

### 四、审计报告结构

1. 执行摘要：整体健康评估 + Top 3-5 优先问题
2. 发现项格式：
- Issue（问题）→ Impact（影响）→ Evidence（证据）→ Fix（修复）→ Priority（优先级）
3. 行动计划：关键修复 → 高影响改进 → 快速胜利 → 长期建议
