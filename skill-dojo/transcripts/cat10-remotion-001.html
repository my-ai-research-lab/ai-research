## Remotion 视频生成核心概念

基于 remotion-best-practices 技能，以下是 Remotion 视频生成的核心概念：

### 1. Composition（组合）
Remotion 的基本单元，定义了一个可渲染视频的关键属性：
- component: 视频内容的 React 组件
- width/height: 视频尺寸
- fps: 帧率
- durationInFrames: 总帧数

<Composition 
  id="MyVideo" 
  component={MyComponent} 
  durationInFrames={100} 
  fps={30} 
  width={1080} 
  height={1080} 
/>


### 2. Frame-Driven Animation（帧驱动动画）
核心原则：所有动画必须通过 useCurrentFrame() 钩子驱动

const frame = useCurrentFrame();
const { fps } = useVideoConfig();
const opacity = interpolate(frame, [0, 2 * fps], [0, 1], {
  extrapolateRight: 'clamp',
});


禁止使用：CSS transitions/animations 和 Tailwind 动画类

### 3. Interpolation（插值）
- interpolate(): 线性插值，将帧映射到任意数值范围
- spring(): 物理弹簧动画，更自然的运动效果
- Easing: 缓动曲线（quad、sin、exp、circle）

// 弹簧动画配置
const smooth = { damping: 200 };      // 平滑无弹跳
const snappy = { damping: 20, stiffness: 200 }; // 快速响应
const bouncy = { damping: 8 };        // 有弹跳


### 4. Sequencing（序列）
控制元素在时间线上的出现时机：

| 组件 | 用途 |
|------|------|
| <Sequence> | 延迟元素出现，可设置持续时间 |
| <Series> | 元素依次播放，无重叠 |
| premountFor | 预加载组件，避免加载延迟 |

<Series>
  <Series.Sequence durationInFrames={45}><Intro /></Series.Sequence>
  <Series.Sequence durationInFrames={60}><MainContent /></Series.Sequence>
</Series>


### 5. 关键特性
- Local Frame: 在 Sequence 内部，useCurrentFrame() 返回局部帧（从0开始）
- calculateMetadata: 动态设置视频时长和属性
- Still: 单帧图片输出
- Folder: 组织管理多个 Composition
