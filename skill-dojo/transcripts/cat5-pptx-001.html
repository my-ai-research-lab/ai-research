## pptx 技能支持的 PPT 操作

### 核心能力矩阵

| 操作类型 | 支持的功能 | 实现方式 |
|---------|-----------|---------|
| 读取/解析 | 文本提取、内容分析 | markitdown |
| 视觉预览 | 缩略图网格生成 | thumbnail.py + Pillow |
| 原始访问 | XML 解包/打包 | unpack.py / pack.py |
| 模板编辑 | 基于模板修改内容 | 参见 editing.md |
| 从零创建 | 无模板新建演示文稿 | PptxGenJS |
| 格式转换 | PPT → PDF → 图片 | LibreOffice + Poppler |

### 详细功能说明

1. 读取与提取
python -m markitdown presentation.pptx  # 文本提取
python scripts/thumbnail.py presentation.pptx  # 视觉预览
python scripts/office/unpack.py presentation.pptx unpacked/  # 解包XML


2. 编辑工作流
- 分析模板 → 解包 → 操作幻灯片 → 编辑内容 → 清理 → 打包

3. 从零创建
- 使用 PptxGenJS（Node.js）生成全新演示文稿
- 支持自定义配色、字体、布局

4. QA 验证
- 内容检查：检测占位符残留（xxxx、lorem）
- 视觉检查：转图片后检测重叠、溢出、间距问题

### 依赖组件

| 组件 | 用途 |
|-----|------|
| markitdown[pptx] | 文本提取 |
| Pillow | 缩略图生成 |
| pptxgenjs | 从零创建 |
| libreoffice | PDF 转换 |
| poppler | PDF 转图片 |

### 与其他技能的关系

- 本技能：PowerPoint 格式(.pptx)的基础操作
- keynote：Apple Keynote 格式(.key)操作
- work-report-ppt：工作汇报 PPT 风格技能（调用本技能输出）
