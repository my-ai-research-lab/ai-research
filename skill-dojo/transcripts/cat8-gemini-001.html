## Gemini Web 生图技能使用说明

### 技能概述

baoyu-danger-gemini-web 是一个通过逆向工程 Gemini Web API 实现的文本/图像生成技能。

⚠️ 重要提示：这是一个使用逆向工程API的"危险"技能，首次使用需要用户明确同意免责声明。

---

### 核心功能

| 功能 | 说明 |
|------|------|
| 文本生成 | 直接输入 prompt 获取文本回复 |
| 图像生成 | 通过 --image 参数生成图片 |
| 图生图 | 通过 --reference 提供参考图，生成变体或描述 |
| 多轮对话 | 通过 --sessionId 保持对话上下文 |

---

### 使用方式

基本命令格式：
bun /path/to/scripts/main.ts [选项] [prompt]


#### 1. 文本生成
bun scripts/main.ts "你的问题"
bun scripts/main.ts --prompt "你的问题" --model gemini-3-flash


#### 2. 图像生成
bun scripts/main.ts --prompt "一只可爱的猫咪" --image cat.png
bun scripts/main.ts --promptfiles prompt.md --image output.png


#### 3. 图生图（Vision 输入）
# 描述图片
bun scripts/main.ts --prompt "描述这张图" --reference image.png

# 基于参考图生成新图
bun scripts/main.ts --prompt "创建变体" --reference ref.png --image out.png


#### 4. 多轮对话
bun scripts/main.ts "记住这个数字: 42" --sessionId session-abc
bun scripts/main.ts "刚才那个数字是什么?" --sessionId session-abc
