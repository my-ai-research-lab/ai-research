## KIM Skills Assistant 使用说明

### 1. 技能定位

这是快手 KIM 平台的通用底层 API 封装，提供与应用无关的基础能力，可被其他上层技能调用（如 linke-kim-message 林克机器人专属能力）。

### 2. 核心能力

| 模块 | 功能 | 触发场景 |
|------|------|---------|
| 📨 消息发送 | 单聊/群聊、MixCard/文本 | "发KIM消息"、"群消息" |
| 📥 消息拉取 | 按时间范围拉取聊天记录 | "拉群消息" |
| 👥 群管理 | 获取群列表、查询群成员 | "查群成员" |
| ✅ 待办 | 创建待办任务 | "创建待办" |
| 📅 日程/会议室 | 查询空闲会议室、创建日程 | "预订会议室" |

### 3. Token 配置

配置优先级：~/.openclaw/openclaw.json > 环境变量 MARIO_KD_MCP_TOKEN

{
  "skills": {
    "entries": {
      "kim-skills-assistant": {
        "enabled": true,
        "env": { "MARIO_KD_MCP_TOKEN": "your_token_here" }
      }
    }
  }
}


### 4. 关键注意事项

- 限流错误码 42900：不保证消息未投递，不自动重试，记录后人工确认
- 批量操作：间隔 ≥1秒 预防限流
- 与 linke-kim-message 的区别：本技能是通用封装（无凭证/无模板），林克专属能力需使用 link-ks-kim-send-message
