# 图像生成提示词

- **设计风格**：AI/ML风格 (ai-ml-style) + 科技感
- **输出路径**：/tmp/ai-robot-coder.jpg

## 完整提示词

生成一张16:9的科技配图。

【风格要求】
AI/ML科技风格，深色背景，简洁现代，带有微妙的渐变和发光效果。

【布局结构】
- 画面中央：一个AI机器人和一个人类程序员并肩坐在工作台前
- 机器人：流线型设计，发光的眼睛，友好的姿态
- 程序员：专注地看着屏幕，手放在键盘上
- 背景：数字代码流、神经网络节点、柔和的蓝色科技光晕

【具体内容】
- AI机器人：白色/银色金属外壳，蓝色发光线条，头部有显示屏或LED眼睛
- 人类程序员：侧面轮廓，穿着休闲，戴着耳机
- 电脑屏幕：显示代码（模糊处理，不需要真实代码）
- 工作环境：现代办公桌，多显示器设置

【视觉细节】
- 整体色调：深蓝色为主，点缀科技蓝和白色
- 光效：机器人身上有柔和的蓝色光晕，屏幕发出的光照亮两者的面部
- 细节：代码元素作为装饰性背景，如0和1、代码括号等
- 氛围：协作、专业、未来感

【禁止元素】
- 禁止生成真实可读的代码
- 禁止出现任何文字或标题
- 禁止过于复杂的细节，保持简洁
