## docx 技能支持的 Word 操作

根据加载的 docx 技能，它支持以下 Word 文档操作：

### 一、读取与分析
| 操作 | 方法 |
|------|------|
| 文本提取（含修订追踪） | pandoc --track-changes=all |
| 原始 XML 访问 | 解包脚本获取底层结构 |
| 转换为图片 | LibreOffice → PDF → 图片 |

### 二、创建新文档（docx-js）
- 页面设置：尺寸（US Letter/A4）、方向（横向/纵向）、边距
- 样式定义：覆盖内置标题样式（Heading1-3）、自定义字体/字号
- 段落格式：对齐、间距、缩进
- 列表：项目符号列表、编号列表（多级嵌套）
- 表格：单元格合并、边框、底纹、内边距、宽度控制
- 图片：嵌入 PNG/JPG/GIF/BMP/SVG，设置尺寸和替代文本
- 页眉页脚：自定义内容、页码
- 目录：自动生成可点击目录
- 分页符：手动分页控制
- 超链接：外部链接

### 三、编辑现有文档（XML 级别）
- 修订追踪：
- 插入标记（<w:ins>）
- 删除标记（<w:del>）
- 拒绝他人插入 / 恢复他人删除
- 删除整个段落/列表项
- 批注：添加批注、回复批注、自定义作者
- 图片替换/插入：通过关系文件管理
- 格式保留：保持原有 <w:rPr> 样式

### 四、格式转换
| 转换 | 支持 |
|------|------|
| .doc → .docx | ✅ LibreOffice |
| .docx → .md | ✅ Pandoc |
| .docx → .pdf | ✅ LibreOffice |
| .docx → 图片 | ✅ PDF 中转 |

### 五、验证与修复
- 自动验证 XML 结构合规性
- 自动修复常见问题（durableId、空白保留属性）
